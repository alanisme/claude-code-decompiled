**Claude Code v2.1.88** に関する独立研究レポート集です。アーキテクチャ、エージェントループ、権限モデル、システムプロンプト、MCP 統合、コンテキスト管理を中心に分析しています。

このリポジトリは意図的に**ドキュメント専用**として構成されています。ここにあるのは分析と論評であり、実行可能なソース配布やサポート対象の CLI ビルドを提供するものではありません。

## このリポジトリの目的

Claude Code は、現在もっとも高度な本番運用向け AI コーディングエージェントの一つです。これらのレポートは、その内部構造を研究しやすくすることを目的としています。

- **エージェント開発者** は、ツールオーケストレーション、コンテキスト管理、回復ロジックの実践パターンを学べます
- **セキュリティ研究者** は、テレメトリ、リモート制御、権限境界を調べられます
- **AI 研究者** は、システムプロンプト構築、モデルルーティング、エージェントループ設計を観察できます
- **エンジニア** は、CLI ベースのエージェントシステム設計の参考資料として使えます

これらのレポートは、**公開配布されている npm パッケージ**の技術分析に基づく研究・論評です。
